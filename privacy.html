Privacy Policy for TikTok Data Extractor

This document governs the privacy policy of my personal script ("the application").

1. Data Collection

The application accesses data from the TikTok API for personal use only. The data collected includes video metrics such as views, likes, comments, and shares, related exclusively to my own TikTok account.

2. Data Usage

The data collected is used solely for the purpose of personal analysis and to create a private performance report.

3. Data Sharing

No data collected by the application will be shared with any third parties. All data remains private and for personal use.

4. Contact

This is a personal, non-commercial project.
